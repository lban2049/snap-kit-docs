# 用户意图

## 目标用户

本文档主要面向以下两类用户：

1. **开发者**：希望集成 Snap Kit 进行网页自动化、爬虫、截图等功能的开发人员
2. **运维人员**：需要部署和维护 Snap Kit 服务的技术人员

## 使用场景

- 开发者首次接触项目时，需要快速了解 Snap Kit 的核心功能和应用场景
- 开发者在集成过程中查阅 API 的使用方法、参数说明和示例代码
- 开发者需要了解如何使用 Crawler Engine 和 SEO Middleware 两个核心包
- 运维人员需要了解如何通过 Docker 或 Blocklet Server 部署 Snap Kit
- 用户在使用过程中遇到问题时，需要查阅配置说明和最佳实践

## 文档侧重点

本文档采用**技术文档 + 使用指南 + API 参考**的综合形式：

- 提供清晰的项目概览，介绍 Snap Kit 的核心价值和主要功能
- 提供详细的快速开始指南，包括本地开发和部署方式
- 提供完整的 API 参考文档，涵盖三个核心模块的使用方法
- 提供架构说明，帮助开发者理解项目的技术栈和组成
- 提供实际应用场景的示例代码和最佳实践
- 提供配置说明和环境变量参考

不需要过多的内部实现细节，重点是如何使用和集成。
